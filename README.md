# Heritage RV Detailing Website

Production website for Heritage RV Detailing, a mobile RV detailing business serving Middle Tennessee.

- Live site: https://www.heritagervdetailing.com/
- Stack: Vite, React, TypeScript, Tailwind CSS
- Production host: Vercel

## Local development

```sh
npm install
npm run dev
```

The dev server defaults to port `8080`.

## Quality checks

```sh
npm run test
npm run build
npm run lint
```

## Deployment

The repository is connected to the existing Vercel project. Production deploys should be made only after Blake approves shipping the scoped client-site change.
